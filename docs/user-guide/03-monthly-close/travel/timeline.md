# Travel Timeline

View your trip expenses on a visual day-by-day timeline.

## Overview

The Travel Timeline provides a chronological view of all expenses during a business trip. See what you spent each day, identify gaps in documentation, and review the trip flow from start to finish.

## Accessing the Timeline

### From Travel Period

1. Open a travel period
2. Click **View Timeline** or **Timeline View**
3. The visual timeline displays

### From Travel List

1. Go to Travel page
2. Click the timeline icon next to any trip
3. Timeline opens for that period

![Timeline view](../../images/travel/timeline-view.png)
*Caption: Travel timeline showing expenses organized by day*

## Timeline Layout

### Horizontal Timeline

The view shows:

- **X-axis**: Days of the trip (start date to end date)
- **Markers**: Each expense appears on its date
- **Height**: Indicates expense amount
- **Color**: Category or type

### Daily Columns

Each day shows:

| Element | Description |
|---------|-------------|
| **Date header** | Day of week, date |
| **Expense cards** | Individual expense summaries |
| **Daily total** | Sum for that day |
| **Category icons** | Visual type indicators |

## Reading the Timeline

### Expense Cards

Each expense card displays:

- Vendor name
- Amount
- Category icon
- Match status indicator
- Time (if available)

### Card Colors

| Color | Meaning |
|-------|---------|
| **Blue** | Transportation |
| **Purple** | Lodging |
| **Orange** | Meals |
| **Green** | Conference/Events |
| **Gray** | Other expenses |

### Click to Expand

Click any expense to see:

- Full details
- Receipt image
- Matching status
- Edit options

## Timeline Features

### Zoom Levels

Adjust the view:

| Level | Shows |
|-------|-------|
| **Days** | Each day in detail (default) |
| **Hours** | Time-of-day breakdown |
| **Overview** | Condensed multi-day view |

### Filtering

Focus on specific expenses:

1. Click **Filter** above the timeline
2. Select categories to show/hide
3. Timeline updates in real-time

### Date Navigation

For longer trips:

- Use arrows to scroll through days
- Click specific dates to jump
- Mini-calendar shows full trip range

## Daily Breakdown

### Day Summary

Each day column includes:

| Metric | Description |
|--------|-------------|
| **Total** | All expenses for the day |
| **Meals** | Food and dining |
| **Transport** | Ground transportation |
| **Other** | All other categories |

### Per Diem Indicator

If per diem applies:

- Green bar: Under daily limit
- Yellow bar: At daily limit
- Red bar: Over daily limit

## Identifying Issues

### Missing Days

Days with no expenses show:

- "No expenses recorded"
- Consider if expenses are missing
- May be legitimate (no spending)

### Missing Receipts

Expenses without receipts show:

- Warning indicator
- "Receipt needed" message
- Click to add receipt

### Unlinked Expenses

Expenses that could belong:

- "Suggested expense" section
- Review and link if appropriate
- Dismiss if not trip-related

## Timeline Actions

### From the Timeline

Quick actions available:

| Action | How |
|--------|-----|
| **Add receipt** | Click + on any day |
| **Link expense** | Drag to day or click to add |
| **View details** | Click any expense |
| **Edit expense** | Click edit icon |

### Bulk Actions

Select multiple:

1. Hold Shift and click
2. Select expenses across days
3. Apply bulk action (categorize, tag, etc.)

## Exporting Timeline

### As Image

1. Click **Export** → **Image**
2. Choose format (PNG/PDF)
3. Timeline exports as visual

### As Data

1. Click **Export** → **Data**
2. Choose format (CSV/Excel)
3. Get tabular data with dates

## Use Cases

### Trip Review

End-of-trip audit:

1. Open timeline
2. Review each day
3. Check for missing receipts
4. Verify all expenses linked

### Expense Justification

For audits or approvals:

1. Export timeline as PDF
2. Shows chronological flow
3. Demonstrates trip pattern
4. Supports expense validity

### Pattern Analysis

Understanding spending habits:

- Which days are highest spend?
- What categories dominate?
- How does this trip compare?

## Best Practices

### Daily Check

During travel:

- Review timeline each evening
- Add missing receipts
- Note any discrepancies

### Complete Documentation

Ensure each expense has:

- ✓ Receipt attached
- ✓ Category assigned
- ✓ Correct date
- ✓ Clear description

### Pre-Report Review

Before generating report:

1. Open timeline view
2. Review entire trip
3. Fill any gaps
4. Verify accuracy

## Troubleshooting

### "Timeline Won't Load"

If timeline is blank:

1. Verify travel period has expenses
2. Check date range is correct
3. Refresh the page

### "Expense on Wrong Day"

If dates are incorrect:

1. Click the expense
2. Edit to correct date
3. It moves to the right day

### "Can't See All Expenses"

If some are hidden:

1. Check filters are not active
2. Verify zoom level shows all days
3. Scroll to see full timeline
