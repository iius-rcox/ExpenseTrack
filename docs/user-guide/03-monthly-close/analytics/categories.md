# Category Breakdown

Analyze spending by expense category.

## Overview

Category breakdown shows how your expenses distribute across different types. Understand which categories consume the most budget and track changes over time.

## Accessing Category View

1. Go to **Analytics** in the main navigation
2. Click the **Categories** tab
3. Or click any category segment in the dashboard pie chart

## Category Chart

### Visualization Types

Toggle between chart styles:

| Style | Best For |
|-------|----------|
| **Pie Chart** | Overall distribution |
| **Donut Chart** | Distribution with total in center |
| **Bar Chart** | Comparing category amounts |
| **Treemap** | Seeing relative sizes |

![Category breakdown](../../images/analytics/category-breakdown.png)
*Caption: Expense categories shown as a donut chart with legend*

### Reading the Chart

Each segment shows:

- Category name
- Total amount
- Percentage of total spending
- Transaction count (on hover)

## Category List

Below the chart, a detailed list shows:

| Column | Description |
|--------|-------------|
| **Category** | Expense category name |
| **Amount** | Total for the period |
| **% of Total** | Share of all spending |
| **Count** | Number of transactions |
| **Avg per Transaction** | Amount / Count |
| **vs. Last Period** | Change indicator |

### Sorting the List

Click column headers to sort:

- By amount (highest first)
- By count (most transactions first)
- By change (biggest increases first)

## Drilling Down

### Category Detail

Click any category to see:

1. **Sub-categories** (if available)
   - Travel → Flights, Hotels, Ground Transport
   - Meals → Client, Team, Solo

2. **Top vendors** in that category
   - Who you spend the most with
   - Useful for negotiating discounts

3. **Transaction list**
   - Individual expenses in that category
   - Click to view details

### Time-Based Drill Down

See category spending over time:

1. Click a category
2. Select **View Trend**
3. Chart shows that category's spending by week/month

## Comparing Categories

### Side-by-Side Comparison

Compare two categories:

1. Click **Compare** button
2. Select first category
3. Select second category
4. View overlaid trend lines

### Period Comparison

Compare a category across periods:

1. Click a category
2. Enable **Compare to Last Period**
3. See this month vs. last month for that category

## Budget Tracking

### Setting Category Budgets

If budgets are configured:

1. Budget line appears on category bar chart
2. Red highlight indicates over-budget
3. Percentage shows actual vs. budget

### Budget Alerts

Categories approaching limits show:

| Status | Meaning |
|--------|---------|
| ✓ Green | Under 75% of budget |
| ⚠ Yellow | 75-100% of budget |
| ✗ Red | Over budget |

## Recategorization Impact

### Seeing Changes

When you [recategorize expenses](../../02-daily-use/transactions/bulk-operations.md):

- Charts update immediately
- Historical data reflects new categories
- Trend comparisons may shift

### Consistent Categorization

For accurate analysis:

- Review AI categorization suggestions
- Use consistent categories for similar vendors
- The AI learns from your [corrections](../../02-daily-use/matching/improving-accuracy.md)

## Common Analysis Tasks

### Finding Savings Opportunities

1. Sort by **Amount** (highest first)
2. Identify top 3 spending categories
3. Review vendors within those categories
4. Look for consolidation opportunities

### Preparing for Budget Review

1. Select the budget period (month/quarter)
2. Export category data to Excel
3. Compare actual vs. planned
4. Annotate significant variances

### Tracking Policy Compliance

1. Filter to policy-relevant categories (Meals, Travel)
2. Review average transaction amounts
3. Flag expenses exceeding limits
4. Document for compliance reports

## Exporting Category Data

### Full Export

1. Click **Export Categories**
2. Choose format:
   - **CSV**: Raw data for analysis
   - **Excel**: Formatted with charts
   - **PDF**: Summary report

### Filtered Export

1. Apply desired filters (date, department, project)
2. Export includes only filtered data
3. Useful for project-specific reporting

## Integration with Reports

Category data feeds into [expense reports](../reports/generating.md):

- Reports include category breakdown
- PDF exports show pie chart
- Excel exports include category columns
