## Monthly Close Checklist

Use this checklist at month-end:

- [ ] Import final bank statement for the month
- [ ] Review and approve all pending match proposals
- [ ] Categorize any unmatched transactions
- [ ] Split shared expenses across departments/projects
- [ ] Create expense report for the month
- [ ] Review subscription alerts
- [ ] Check analytics for unusual spending patterns

## Typical Monthly Workflow

```
End of Month
     ↓
Import final statement
     ↓
Review & approve matches
     ↓
Generate expense report
     ↓
Review analytics
     ↓
Submit report for approval
```
