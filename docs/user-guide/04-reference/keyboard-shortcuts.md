# Keyboard Shortcuts Reference

← [Back to Reference](./README.md)

Quick reference card for all ExpenseFlow keyboard shortcuts.

## Overview

This is a **condensed quick-reference card** for printing or quick lookup. For detailed explanations of each shortcut with context and tips, see the [comprehensive keyboard shortcuts guide](../02-daily-use/keyboard-shortcuts.md).

## Global Navigation

| Shortcut | Action |
|----------|--------|
| `?` | Show shortcuts help |
| `G` `D` | Go to Dashboard |
| `G` `R` | Go to Receipts |
| `G` `T` | Go to Transactions |
| `G` `M` | Go to Matching |
| `G` `A` | Go to Analytics |
| `/` | Focus search |
| `Esc` | Close/Cancel |

## Receipt Viewer

### Zoom
| Shortcut | Action |
|----------|--------|
| `+` / `=` | Zoom in |
| `-` | Zoom out |
| `0` | Fit to window |
| `1` | Actual size (100%) |

### Transform
| Shortcut | Action |
|----------|--------|
| `R` | Rotate right |
| `Shift+R` | Rotate left |
| `F` | Fullscreen |

### Navigate
| Shortcut | Action |
|----------|--------|

## Notes

- **Mac users**: Use `Cmd` instead of `Ctrl`
- Press `?` anywhere to see available shortcuts for current screen
- Shortcuts can be disabled in Settings → Accessibility
