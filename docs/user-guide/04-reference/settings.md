# Settings

← [Back to Reference](./README.md)

Customize ExpenseFlow to match your preferences and workflow.

## Overview

The Settings page lets you personalize your ExpenseFlow experience. Adjust themes, set defaults, manage templates, and configure notifications.

## Accessing Settings

1. Click your **Profile** icon in the top-right corner
2. Select **Settings**
3. Or use keyboard shortcut `G` then `S`

![Settings full view](../images/settings/settings-full-view.png)
*Caption: Settings page showing all configuration sections*

## Settings Sections

### Appearance

Control how ExpenseFlow looks.

#### Theme

Choose your visual theme:

| Option | Description |
|--------|-------------|
| **Light** | Bright background, dark text |
| **Dark** | Dark background, light text |
| **System** | Matches your OS preference |

![Theme selector](../images/settings/theme-selector.png)
*Caption: Theme options with light/dark/system choices*

To change theme:

1. Go to Settings → Appearance
2. Click your preferred theme
3. Changes apply immediately

#### Accent Color

Some themes support accent colors:

- Choose from preset colors
- Or use company brand color
- Applied to buttons and highlights

### Defaults

Set standard values for common fields.

#### Default Department

Set your primary department:

1. Go to Settings → Defaults
2. Select from Department dropdown
3. New expenses auto-assign to this department

#### Default Project

Set your current project:

- Used when creating new expenses
- Can be overridden per expense
- Useful for project-based work

#### Default Category

Optionally set a default GL code:

- Applied to uncategorized expenses
- Consider leaving blank for AI suggestions
- Useful if most expenses are one type

### Fingerprint Templates

Manage saved [fingerprints](../02-daily-use/statements/fingerprints.md) for statement imports.

#### Viewing Templates

1. Go to Settings → Fingerprints
2. See list of all saved templates
3. Each shows name, bank/source, and last used

#### Editing Templates

1. Click a template name
2. Modify mappings if needed
3. Save changes

#### Deleting Templates

1. Click the delete icon (trash)
2. Confirm deletion
3. Template removed (existing imports unaffected)

### Notifications

Control what alerts you receive.

#### Email Notifications

| Setting | Description |
|---------|-------------|
| **Weekly summary** | Pending actions digest |
| **Report updates** | Approval/rejection notices |
| **System alerts** | Maintenance and outages |

#### In-App Notifications

| Setting | Description |
|---------|-------------|
| **Match proposals** | New matches ready for review |
| **Subscription alerts** | Renewal and change notices |
| **Processing complete** | When receipts finish processing |

#### Notification Frequency

Control how often you're notified:

- **Immediately**: Real-time alerts
- **Hourly**: Batched every hour
- **Daily**: Once per day digest
- **Weekly**: Weekly summary only

### Privacy & Data

Manage your data and privacy settings.

#### Data Retention

View data retention policies:

- How long receipts are stored
- When data is archived
- Export options before deletion

#### Export Your Data

Request a data export:

1. Click **Request Export**
2. Choose format (JSON or CSV)
3. Receive download link via email
4. Download within 7 days

#### Delete Account Data

For account deletion:

1. Contact your administrator
2. All personal data will be removed
3. Receipts may be retained for compliance

### Accessibility

Configure accessibility options.

#### Keyboard Shortcuts

- **Enable/Disable**: Toggle all shortcuts
- **View shortcuts**: See full reference
- Useful if shortcuts conflict with assistive tech

#### Reduced Motion

Minimize animations:

- Reduces page transitions
- Simplifies progress indicators
- Helpful for motion sensitivity

#### High Contrast

Enable high contrast mode:

- Increased color contrast
- Clearer focus indicators
- Better for low vision users

### Connected Accounts

Manage integrations and connected services.

#### Bank Connections

If bank integration is enabled:

- View connected accounts
- Refresh connection status
- Disconnect accounts

#### Calendar Integration

For expense scheduling:

- Connect calendar for reminders
- Sync report deadlines
- Travel period suggestions

### Advanced

Additional settings for power users.

#### Date Format

Choose your preferred format:

| Format | Example |
|--------|---------|
| MM/DD/YYYY | 12/31/2025 |
| DD/MM/YYYY | 31/12/2025 |
| YYYY-MM-DD | 2025-12-31 |

#### Currency Display

Set display preferences:

- Symbol position (before/after)
- Decimal separator
- Thousands separator

#### Time Zone

Set your time zone:

- Auto-detect from browser
- Manual selection
- Affects displayed dates/times

## Saving Settings

- Most settings save automatically
- Look for "Saved" confirmation
- Some require page refresh

## Resetting to Defaults

To reset all settings:

1. Scroll to bottom of Settings page
2. Click **Reset to Defaults**
3. Confirm the reset
4. All settings return to original values

## Settings Not Working?

If settings don't appear to save:

1. Check your internet connection
2. Refresh the page
3. Try logging out and back in
4. Clear browser cache
