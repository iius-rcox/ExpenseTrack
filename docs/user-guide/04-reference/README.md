## Quick Troubleshooting

**Can't upload a receipt?**
- Check file size (max 25MB)
- Verify format (JPEG, PNG, HEIC, PDF)
- See [Upload Troubleshooting](./troubleshooting.md#upload-fails-with-file-too-large)

**Match proposals not appearing?**
- Run **Auto-Match** from the Matching page
- Ensure you have both receipts and transactions
- See [Matching Troubleshooting](./troubleshooting.md#no-match-proposals-appearing)

**Session expired?**
- Sign in again at the login page
- Check [Session Issues](./troubleshooting.md#session-expired-during-upload)

## Keyboard Shortcut Summary

| Key | Action |
|-----|--------|
| `A` | Approve match |
| `R` | Reject match |
| `J` / `↓` | Next item |
| `K` / `↑` | Previous item |
| `M` | Manual match |
| `?` | Show shortcuts help |
| `Esc` | Close dialog/help |

See [Keyboard Shortcuts](./keyboard-shortcuts.md) for the complete list.

## Common Terms

| Term | Definition |
|------|------------|
| **Confidence Score** | AI certainty percentage (green 90%+, amber 70-89%, red <70%) |
| **Fingerprint** | Saved column mapping template for statement imports |
| **Match Proposal** | AI-suggested link between receipt and transaction |
