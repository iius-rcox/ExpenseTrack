## Typical Daily Workflow

```
1. Upload receipts from your phone/computer
         ↓
2. Import bank statement (weekly/monthly)
         ↓
3. Review match proposals
         ↓
4. Categorize any unmatched transactions
         ↓
5. Split shared expenses if needed
```
