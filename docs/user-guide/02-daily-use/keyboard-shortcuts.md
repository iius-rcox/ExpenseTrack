# Keyboard Shortcuts

← [Back to Daily Use](./README.md) | [Quick Reference Card](../04-reference/keyboard-shortcuts.md)

Work faster with keyboard shortcuts throughout ExpenseFlow.

## Overview

Power users can significantly speed up their workflow using keyboard shortcuts. This **comprehensive guide** covers shortcuts for navigation, match review, receipt viewing, and common actions with detailed context and tips.

> **Quick lookup?** See the [printable quick reference card](../04-reference/keyboard-shortcuts.md) for a condensed version.

## Global Shortcuts

Available anywhere in the app:

| Shortcut | Action |
|----------|--------|
| `?` | Show shortcuts help panel |
| `G` then `D` | Go to Dashboard |
| `G` then `R` | Go to Receipts |
| `G` then `T` | Go to Transactions |
| `G` then `M` | Go to Matching |
| `G` then `A` | Go to Analytics |
| `/` | Focus search box |
| `Esc` | Close modal or cancel action |

> **Tip**: Press `?` anywhere to see a quick reference panel of available shortcuts.

## Match Review Shortcuts

When in the [Review Mode](./matching/review-modes.md) for matching:

### Navigation

| Shortcut | Action |
|----------|--------|
| `J` | Next match proposal |
| `K` | Previous match proposal |
| `↓` | Next match (alternative) |
| `↑` | Previous match (alternative) |
| `Enter` | Open selected match details |

### Actions

| Shortcut | Action |
|----------|--------|
| `A` | Approve current match |
| `R` | Reject current match |
| `S` | Skip to next (no action) |
| `U` | Undo last action |
| `M` | Open manual match dialog |
| `E` | Edit match details |

### Batch Operations

| Shortcut | Action |
|----------|--------|
| `Shift+A` | Approve all high-confidence |
| `Ctrl+A` / `Cmd+A` | Select all visible matches |
| `Space` | Toggle selection on current item |

## Receipt Viewer Shortcuts

When viewing a receipt image:

### Navigation

| Shortcut | Action |
|----------|--------|
| `←` | Previous receipt |
| `→` | Next receipt |
| `Home` | First receipt |
| `End` | Last receipt |

### Image Controls

| Shortcut | Action |
|----------|--------|
| `+` or `=` | Zoom in |
| `-` | Zoom out |
| `0` | Reset zoom to fit |

| Shortcut | Action |
|----------|--------|
| `E` | Edit extracted fields |
| `Tab` | Next field (when editing) |
| `Shift+Tab` | Previous field (when editing) |
| `Enter` | Save changes |
| `Esc` | Cancel editing |

## Accessibility Notes

### Screen Reader Support

All shortcuts have corresponding button alternatives for accessibility. Screen reader users can:

- Use Tab to navigate between actions
- Press Enter to activate buttons
- Use arrow keys in lists

### Disabling Shortcuts

If shortcuts conflict with assistive technology:

1. Go to **Settings** → **Accessibility**
2. Toggle **Keyboard Shortcuts** off
3. All actions remain available via buttons/menus

## Customizing Shortcuts

Currently, shortcuts cannot be customized. Default shortcuts are designed to be familiar to users of similar applications.
