# Example Data

This folder contains sample expense reports, receipts, and statements for testing ExpenseFlow.

## Contents (not tracked in git)

- `Expense Reports/` - Sample expense report PDFs and receipts organized by month
- Receipts (JPG, PNG, PDF)
- Submitted reports (PDF, XLSX)

## Why not tracked?

The actual data files are excluded from git to:
- Protect sensitive financial information
- Keep repository size manageable
- Allow each developer to use their own test data

## Setup

Place your own test data in this folder following the structure above.
